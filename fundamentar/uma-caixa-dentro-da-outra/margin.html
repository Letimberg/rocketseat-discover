Margin
Elemento abaixo

Margin, é o espaço (margem) entre os elementos

Podemos dividir o margin em 4 valores

  1. Margin-top
  2. Margin-right
  3. Margin-bottom
  4. Margin-left
    • Values: < length >
    • Percentage: auto >

Geralmente usamos uma forma abreviada (shorthand) para escrever o margin. Esse formato segue o sentido horário iniciando pelo top, seguindo para right, bottom e left.

  1. margin: 12px 16px 10px 4px; /* TOP = 12px | RIGHT = 16px | BOTTOM = 10px | LEFT = 4px */
  2. margin: 12px 16px 0; /* TOP = 12px | RIGHT = 16px | BOTTOM = 0px | LEFT = 16px */
  3. margin: 8px 16px; /* TOP = 8px | RIGHT = 16px | BOTTOM = 8px | LEFT = 16px */
  4. margin: 8px; /* TOP = 8px | RIGHT = 8px | BOTTOM = 8px | LEFT = 8px */

O margin é aplicado em elementos com display block

Cuidado com o margin collapsing que é quando o top se junta ao bottom
